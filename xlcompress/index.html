Output mode ? Aggregated — groups cells of the same data type into labeled regions, replacing values with type tags like IntNum or DateData.

Vanilla — lists each non-empty cell as |Address, Value|.

Inverted Index — maps each unique value to its list of cell addresses.
Options ? Structural compression — detects table anchor rows and columns, then removes rows and columns far from any anchor, shrinking the grid before aggregation runs.
